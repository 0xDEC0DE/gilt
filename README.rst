gilt - A GIT layering tool.

Quick Start
===========

Install gilt using pip:

.. code-block:: bash

  $ pip install gilt-python

Overlay remote repositories and/or files into the destinations provided.

.. code-block:: bash

  $ gilt --debug overlay
